# Entendimento Completo do Projeto - Convite Especial

## O que é o projeto

Uma **página web de convite** para um **Reencontro da Faculdade de Enfermagem (turma 2006.2 da UPE)**, agendado para **11 de julho de 2026, das 15h às 20h**. O evento acontece no **Salão de Festas do Edifício Elizabeth Agra**, Rua Jorge de Albuquerque, 142 - Casa Forte, Recife/PE.

O projeto é um **arquivo HTML único** (`index.html`) com CSS e JavaScript inline, sem frameworks, sem bundler, sem dependências externas além do Google Fonts.

---

## Análise do `index.html` (detalhada)

### **Linha 1-8: Cabeçalho HTML**
- Lang `pt-BR`, charset UTF-8, viewport responsivo
- Título: "Reencontro Faculdade de Enfermagem - 11/07/2026"
- Google Fonts: **Playfair Display** (títulos serifados) + **Montserrat** (corpo sans-serif)

### **Linha 8-298: CSS (inline)**
- **Paleta de cores**:
  - `#F7F4EB` — fundo creme/bege claro
  - `#1D2D44` — azul escuro (texto principal, cards escuros)
  - `#C5A059` — dourado (destaques, bordas, ícones, badges)
  - `#9AB3B5` — cinza-azulado claro (textos secundários)

- **Seções CSS**:
  - **Hero** (`.hero`): Tela inteira, gradiente radial sutil com animação shimmer, partículas flutuantes
  - **Countdown** (`.countdown-section`): Fundo escuro (#1D2D44), cards com borda dourada
  - **Event Cards** (`.event-grid`): Grid responsivo de 4 cards (Data, Horário, Local, Estilo)
  - **Gallery** (`.gallery-grid`): Grid responsivo de imagens com hover scale e modal
  - **Map** (`.map-section`): Container com borda dourada para iframe Google Maps
  - **Footer** (`.footer`): Fundo escuro com frase e crédito "DESYSIS"
  - **Modal** (`.modal`): Overlay escuro para visualização de fotos em tamanho grande
  - **Reveal Animation** (`.reveal`): Animação de entrada ao scroll (fade-in + slide-up)
  - **Responsivo**: Breakpoints em 380px, 600px, 900px, 1200px

### **Linha 300-357: HTML Structure**
1. **Hero Section**: Badge "CONVITE ESPECIAL" + logo do evento + slogan
2. **Countdown Section**: Contagem regressiva (Dias, Horas, Minutos, Segundos)
3. **Event Section**: 4 cards com detalhes do evento + mensagens (comentadas)
4. **Map Section**: Iframe do Google Maps com localização do evento
5. **Gallery Section**: "Momentos Inesquecíveis" — grid de fotos
6. **Footer**: Citação + crédito "Com carinho para toda a turma de Enfermagem 2006.2" + "DESYSIS"
7. **Modal**: Para visualização ampliada de fotos

### **Linha 359-463: JavaScript (inline)**

- **Partículas** (360-369): Cria 40 partículas douradas flutuantes com positions/delays aleatórios
- **Countdown Timer** (371-381): Conta regressiva até `2026-07-11T15:00:00`, atualiza a cada segundo
- **Scroll Reveal** (383-385): Intersection observer simplificado — element aparece ao entrar no viewport
- **Gallery Builder** (387-458):
  - Array `mediaFiles` com **48 nomes de arquivo** (todos `.jpeg` do WhatsApp)
  - Cria dinamicamente os elementos da galeria
  - Suporta imagens e vídeos (`.mp4`, `.webm`, `.ogg`)
  - Clique abre modal com a foto ampliada
- **Modal** (460-462): Fecha com botão X, clique fora, ou tecla Escape

## Observações Importantes

1. **Não há vídeos** na galeria — apenas imagens JPEG
2. **Não há framework** — é HTML puro, CSS puro, JS puro
3. **Não há build system** — basta abrir `index.html` no navegador
4. **Não há backend** — é um site estático
5. **Não há analytics, formulários ou interatividade complexa**
6. **O site é responsivo** — funciona em mobile e desktop
7. **Google Maps embutido** — localização do evento
8. **Citação no footer**: "Cada um sabe a dor e a delícia de ser o que é..." (Chico Buarque)
9. **Crédito**: "DESYSIS" (provavelmente o criador do site)
10. **Git commits**: 5 commits (first commit → zoom updates → emoji update)
